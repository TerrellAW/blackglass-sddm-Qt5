# BlackGlass SDDM Theme (Qt5/Plasma 5)

Qt5-compatible version of phob1an's BlackGlass SDDM theme.

## About

This is a backport of the [original BlackGlass theme](https://www.opencode.net/phob1an/blackglass) by phob1an, modified to work with Qt5 and Plasma 5.

### Changes Made
- Updated QML imports from Qt6 to Qt5 syntax
- Fixed arrow function syntax for Qt5 compatibility
- Modified Main.qml and ComboBox.qml
- Added new DigitalSegmented font

## Installation

1. Clone or download repo to: /usr/share/sddm/themes/

2. Activate through KDE Plasma's Startup and Shutdown settings
